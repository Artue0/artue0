ABOUT THIS
-WEBSTIE-

This is a website I made for fun, and also because my old one is really bad and only about cats.

Warning! This website acts unintentionally on narrow displays like phones. I coudl fix it, but I'm too lazy for that.

The 3D part of this website is made with a library called "Three.js", which uses WebGL to render interactive 3D graphics directly within web browsers.

All of the source code is available on my github but I highly do not recommend you seeing it, it's literally a mess. You woudln't understand anything, I don't even know what's going on there.

This website took me a long time to make, so I hope you like it!
